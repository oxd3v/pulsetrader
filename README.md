## ğŸ“– Overview

PulseTrader is a sophisticated multi-chain cryptocurrency trading terminal built with Next.js. It provides real-time market analytics, advanced order management, and portfolio tracking across multiple blockchain networks including Ethereum, Avalanche, Arbitrum, and Solana.

The platform enables users to execute spot trades with advanced features like grid trading, technical entry/exit conditions, stop-loss, take-profit, and re-entrance strategies.

## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm, yarn, pnpm, or bun
- MetaMask browser extension

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-repo/pulsetrader.git
   cd pulsetrader
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Configure environment variables**
   
   Create a `.env.local` file:
   ```env
   NEXT_PUBLIC_AUTH_TOKEN_SECURITY_PASSWORD=your-encryption-password
   NEXT_PUBLIC_API_URL=your-backend-api-url
   ```

4. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

5. **Open in browser**
   
   Navigate to [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
npm run start
```

## ğŸ” Security Considerations

2. **Auth Tokens**: Encrypted with AES before storage
